## Теорема Виета

Для квадратного уравнения ax² + bx + c = 0:
- **Сумма корней**: r₁ + r₂ = -b/a
- **Произведение корней**: r₁ × r₂ = c/a

Для приведенного вида x² + px + q = 0:
- **Сумма корней**: r₁ + r₂ = -p  
- **Произведение корней**: r₁ × r₂ = q

## Алгоритм быстрого решения (для ЕГЭ)

1. **Привести к виду** x² + px + q = 0
2. **Найти делители** свободного члена q
3. **Подобрать пару** с суммой = -p
4. **Проверить** (при необходимости)

### Пример: x² - 5x - 6 = 0
- Сумма корней = 5, произведение = -6
- Делители -6: ±1, ±2, ±3, ±6
- Пары: (6,-1) → сумма 6+(-1) = 5 ✓
- **Ответ: корни 6 и -1**
